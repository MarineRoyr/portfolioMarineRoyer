A propos

Après l'obtention d'un titre RNCP développeur web, je suis àla recherche d'un poste de développeur junior / intégrateur web. afin de mettre en pratique mes connaissances et évoluer sur le terrain.

J'ai de solides bases sur les langages basiques tels que HTML, CSS et Javascript, je travaille sur React et Node.js. Je maîtrise les bases de Git, GitHub et Gitflow. Anciennement graphiste, j'ai une certaine appétence pour le frontend et l'UX Design.

Je suis mobile sur tout l'est francilien et à Paris.

Après une carrière de fleuriste évènementielle et un an d'expérience dans la communication d'entreprise & le community management, je me consacre entièrement à mon objectif professionnel autour du développement web.
